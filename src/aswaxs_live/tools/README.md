# FrameByFrame Tools

- `iq_viewer`: I-q plotting, metadata inspection, subtraction, export, and QC
- `online_reducer`: ZMQ-driven online 1-D SAXS/WAXS reduction
- `pyfai_setup`: PONI, mask, and radial-integration setup
- `rack_builder`: sample, solvent, GC, air, and empty-position planning
- `linkers`: adapters to external XAnoS, XModFit, and sample-position apps

External application source code remains outside FrameByFrame-ASWAXS.
